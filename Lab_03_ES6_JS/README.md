# Lab 03 — ES6 JavaScript

This lab focuses on applying ES6+ JavaScript concepts through seven standalone practical tasks.

## Folder Structure

```
Lab_03_ES6_JS/
├── index.html
├── style.css
├── script.js
├── task1/
├── task2/
├── task3/
├── task4/
├── task5/
├── task6/
└── task7/
```

## Tasks Overview

### Task 1 — Student Management
Path: `task1/`

- Build and manage student records.
- Practice object/array handling and dynamic rendering.

### Task 2 — Shopping Cart
Path: `task2/`

- Simulate cart operations such as adding/removing items.
- Work with totals and list updates in the DOM.

### Task 3 — Async Data Loader
Path: `task3/`

- Load data asynchronously.
- Practice Promise / async flow and user feedback states.

### Task 4 — Course Registration System
Path: `task4/`

- Register learners in courses with validation.
- Manage state and updates interactively.

### Task 5 — Product Catalog
Path: `task5/`

- Display and manage a list of products.
- Apply filtering/sorting style logic and reusable rendering.

### Task 6 — Mini University Portal
Path: `task6/`

- Combine multiple entities (students, courses, etc.) in one mini system.
- Practice modular thinking and structured UI logic.

### Task 7 — Student Data with JSON
Path: `task7/`

- Work with JSON-based data structures.
- Parse and present structured data dynamically.

## How to Run

1. Open `index.html` for the main Lab 03 landing page, or open any task's `index.html` directly.
2. Use a modern browser (Chrome, Firefox, Edge).

## Skills Practiced

- ES6 syntax and best practices
- Array/object operations
- Functions and modular logic
- Async JavaScript patterns
- DOM manipulation
