# Lab 04 — ES6 + jQuery Interactive Labs

This lab contains eight jQuery-focused interactive tasks, accessible from a dashboard-style landing page.

## Folder Structure

```
Lab_04_ES6_JS.(JQuery)/
├── index.html
├── css/
│   └── main.css
├── js/
│   └── main.js

    ├── task1/
    ├── task2/

    └── task8/
```

## Tasks Overview

### Task 1 — Dynamic List Manager
Path: `tasks/task1/`

- Add/remove/manage list items.
- Includes animation and input checks.

### Task 2 — Animated Image Gallery
Path: `tasks/task2/`

- Browse images with transitions.
- Includes controls and smooth visual updates.

### Task 3 — Interactive Form Validation
Path: `tasks/task3/`

- Validate fields in real time.
- Show inline feedback and submit-state handling.

### Task 4 — Tabbed Content Interface
Path: `tasks/task4/`

- Switch between content tabs with interactive transitions.
- Supports cleaner navigation patterns.

### Task 5 — Chained Style Editor
Path: `tasks/task5/`

- Update CSS styles through jQuery chaining.
- Demonstrates direct style manipulation and live preview.

### Task 6 — API Data Fetcher
Path: `tasks/task6/`

- Fetch and display data from an API.
- Covers loading, rendering, and error handling.

### Task 7 — Drag-and-Drop Sortable
Path: `tasks/task7/`

- Reorder items using drag-and-drop behavior.
- Useful for list prioritization workflows.

### Task 8 — Knowledge Quiz
Path: `tasks/task8/`

- Interactive quiz experience with scoring.
- Provides immediate user feedback.

## How to Run

1. Open `index.html` in this folder to access the dashboard.
2. Click **Open Lab** on any task card.
3. You can also open each task's `index.html` directly.

## Skills Practiced

- jQuery selectors and events
- DOM manipulation and animation
- Form handling and validation
- Asynchronous data loading
- UI/UX interaction patterns
